Custom File system 
__________________

- Idea of the project is to build a restfull application which adhare to 

	2nd level of richardson maturity model 
	- usage of currect HTTP method considering idempotence behavior.
        - JAX-RS jersey library used

 -SOLID Object oriented design and analysis.
 		- Proper encapsulation is done to maintain consistent source of change.
 		- Factory pattern and open close principle is utilse to achieve dependency inversion.
 		- FileEntity implemented Interface  segregation principle to achieve request bequest.
 		- proper usage of enum is encouraged.

- Data structure implemented in n-array tree to keep a file entiry nodes.
